# ARRMA Typhon 1/8 3S Reference Specs

Raw manufacturer reference for the ARRMA Typhon 1/8 3S platform used by the Turbo port.

## Source

- ARRMA product page:
  - <https://www.arrma-rc.com/de/product/1-8-typhon-3s-4x4-rtr-brushless-buggy-black/ARA102696.html>

## Notes

- This page intentionally records the manufacturer-published values directly.
- These values are not automatically the same as the runtime `CarSpecs` used in the Turbo port.
- The Turbo port may use a different effective mass if it models the full running system rather than the stock RTR vehicle.
